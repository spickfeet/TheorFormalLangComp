Чтобы создать новый текстовый файл, нажмите "Файл" -> "Создать".
Чтобы открыть текстовый файл, нажмите "Файл" -> "Открыть".
Чтобы сохранить изменения в файле, нажмите "Файл" -> "Сохранить".
Чтобы сохранить файл в другой директории, нажмите "Файл" -> "Сохранить как".
Чтобы отменить действие, нажмите "Правка" -> "Отменить".
Чтобы вернуть изменения, нажмите "Правка" -> "Вернуть".
Чтобы скопировать в буфер обмена, нажмите "Правка" -> "Копировать".
Чтобы скопировать в буфер обмена и удалить с текущего места выделения, нажмите "Правка" -> "Вырезать".
Чтобы вставить из буфера обмена, нажмите "Правка" -> "Вставить".
Чтобы стереть всю информацию, нажмите "Правка" -> "Удалить всё".
