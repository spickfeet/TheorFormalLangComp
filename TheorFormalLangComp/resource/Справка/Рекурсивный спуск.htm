Грамматика G[E]:


E => TA
A => e | + TA | - TA
T => OB
B => e | *OB | / OB
O => num | id | (E)

num - числовая константа Ц{Ц}
id - идентификатор Б{Б|Ц}
Б - [A, B, …, Z], Ц - [0, 1, …, 9]

Язык

L(G[Е]) = {x ∈ VT* | E =>* x }
VT = {num, id, +, -, , /, (, )}
VN = {E, A, T, B, O}
Е - начальный нетерминал

Тестовый пример

Выражение: 20+7-d
Последовательность разбора: E--T--O--B--A--T--O--B--A--T--O--B--A--

Классификация

Согласно классификации Хомского, грамматика G[E] является контекстно-свободной.
Все правила относятся к классу продукций: A => a, A ∈ Vn, а ∈ V*
